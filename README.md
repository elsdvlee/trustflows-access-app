# TrustFlows Access Request Tool

## 📌 Overzicht

Standalone React-app om expliciet **TrustFlows access requests** te doen voor één of meerdere resources.  
Handig voor debugging, testing en integratie naast tools zoals Comunica/Miravi.

---

## ⚙️ Functionaliteit

- 🔐 Login via TrustFlows (OIDC)
- 📝 Input van meerdere URLs (één per lijn)
- 🚀 Access request triggeren
- 📊 Resultaat per URL:
  - ✅ granted
  - ❌ denied
  - 🔒 unauthorized
  - ⚠️ error
